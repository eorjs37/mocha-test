# mocha test

## node version 20.12.0

## 명령어

```bash
mocha -w src/파일명
```

## 패턴

- 템플릿 메서드

```
여러 클래스에서 쓰이는 메소드를 공통화하여,상위클래스에 정의하고 하위클래스에서 구현하도록 하는것
```

- 전략 패턴

```
동일 계열의 알고리즘을 정의하고,
각 알고리즘을 캡슐화하여,
상호교환 가능하도록 만든다.
```

- 상태 패턴

```
객체의 상태를 조건문으로 검사해서 행위를 달리하는것이 아닌
상태를 객체화 하여 상태가 행동할 수 있도록 위임하는 패턴.
객체의 상태에 따라 행동을 변경
```

- 전략패턴 상태패턴 차이

```
전략패턴은 알고리즘을 객체화 하는것
상태패턴은 객체의 상태를 객체화 처리
```
